# Flash Loans

Decentralized application for flash loan arbitrage between DEXs.

**Status:** Experimental  
**Use Case:** DeFi arbitrage

⚠️ **Warning:** Experimental software. Use at your own risk.

## What are Flash Loans?

Flash loans allow you to borrow assets without collateral, as long as the loan is repaid within the same transaction.

### Use Cases

- **Arbitrage:** Buy low on one DEX, sell high on another
- **Collateral Swap:** Change collateral in lending protocols
- **Liquidations:** Liquidate undercollateralized positions

## Smart Contract Example

```solidity
contract FlashLoanArbitrage is FlashLoanReceiverBase {
    function executeOperation(
        address[] calldata assets,
        uint256[] calldata amounts,
        uint256[] calldata premiums,
        address initiator,
        bytes calldata params
    ) external override returns (bool) {
        // Execute arbitrage
        _executeArbitrage(assets[0], amounts[0]);
        
        // Approve repayment
        uint256 amountOwed = amounts[0] + premiums[0];
        IERC20(assets[0]).approve(address(LENDING_POOL), amountOwed);
        
        return true;
    }
}
```

## Finding Opportunities

```typescript
async function findArbitrage() {
  const uniswapPrice = await getUniswapPrice('WETH', 'DAI')
  const sushiswapPrice = await getSushiswapPrice('WETH', 'DAI')
  
  const priceDiff = Math.abs(uniswapPrice - sushiswapPrice)
  const diffPercent = (priceDiff / uniswapPrice) * 100
  
  if (diffPercent > 1) {
    return {
      profitable: true,
      buyDex: uniswapPrice < sushiswapPrice ? 'Uniswap' : 'Sushiswap',
      sellDex: uniswapPrice < sushiswapPrice ? 'Sushiswap' : 'Uniswap'
    }
  }
}
```
