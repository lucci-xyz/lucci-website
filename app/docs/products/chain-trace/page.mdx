# Chain Trace

Blockchain analysis tool for NFT transaction histories and visualizations.

**Status:** Research  
**Chain:** Ethereum  

## Overview

Chain Trace helps researchers and collectors understand how NFTs move through the Ethereum ecosystem by analyzing transaction histories.

### Features

- Transaction tracking from mint to current owner
- Wallet behavior analysis
- Collection insights and statistics
- Interactive network visualizations

## Installation

```bash
npm install chain-trace
```

## Quick Start

```typescript
import { ChainTrace } from 'chain-trace'

const trace = new ChainTrace({
  rpcUrl: process.env.ETHEREUM_RPC_URL,
  apiKey: process.env.ETHERSCAN_API_KEY
})

// Get NFT history
const nft = await trace.getNFTHistory({
  contract: '0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D',
  tokenId: '1'
})

console.log(`Traded ${nft.transferCount} times`)
console.log(`Current owner: ${nft.currentOwner}`)
```

## Use Cases

### Due Diligence

Check NFT history before purchasing:

```typescript
const history = await trace.getNFTHistory({
  contract: '0x...',
  tokenId: '1234'
})

if (history.washTradingScore > 0.7) {
  console.warn('Potential wash trading detected')
}
```

### Collection Analysis

Study market behavior:

```typescript
const collection = await trace.analyzeCollection({
  contract: '0x...',
  startDate: '2024-01-01',
  endDate: '2024-12-31'
})

console.log(`Total volume: ${collection.totalVolume} ETH`)
console.log(`Unique traders: ${collection.uniqueTraders}`)
```

## Visualizations

```typescript
import { NetworkGraph } from 'chain-trace/viz'

const data = await trace.getTransactionNetwork({
  contract: '0x...',
  depth: 2
})

const graph = new NetworkGraph(data)
graph.render('#chart', {
  nodeSize: 'nftCount',
  nodeColor: 'totalValue'
})
```
