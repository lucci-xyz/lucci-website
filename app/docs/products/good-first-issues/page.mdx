# Good First Issues

Curated platform for discovering beginner-friendly issues from Web3, Crypto, and DeFi projects.

**Status:** Live  
**Website:** [good-first-issues-three.vercel.app](https://good-first-issues-three.vercel.app/)

## Overview

Good First Issues helps newcomers find their first open-source contribution by aggregating and categorizing beginner-friendly issues from leading Web3 projects.

### Features

- Curated issues from high-quality projects
- Filter by language and technology
- Difficulty ratings
- Real-time updates

## Using the Platform

Visit the [platform](https://good-first-issues-three.vercel.app/) and filter by:
- **Language:** Solidity, Rust, TypeScript, Python
- **Project Type:** DeFi, NFT, Infrastructure
- **Status:** Open, Recently Updated

## For Project Maintainers

Want your project featured? Your project should:
- Be open source and Web3-related
- Have contribution guidelines
- Use the `good first issue` label
- Have active maintainers

### Add Your Project

```bash
git clone https://github.com/natalie-a-1/GoodFirstIssues
cd GoodFirstIssues

# Add to src/config/projects.json
{
  "name": "Your Project",
  "github": "owner/repo",
  "description": "Brief description",
  "technologies": ["typescript", "solidity"]
}

# Submit PR
git add .
git commit -m "Add YourProject"
git push origin add-your-project
```

## API Access

```bash
# Get all issues
curl https://api.goodfirstissues.dev/issues

# Filter by technology
curl https://api.goodfirstissues.dev/issues?tech=solidity
```
