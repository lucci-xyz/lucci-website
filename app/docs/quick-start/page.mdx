# Quick Start

Get up and running with Lucci Labs products.

## BountyPay

Install and use BountyPay for automated bounty payments.

### Installation

```bash
npm install @lucci/bountypay-sdk
```

### Basic Usage

```typescript
import { BountyPayClient } from '@lucci/bountypay-sdk'

const client = new BountyPayClient({
  apiKey: process.env.BOUNTYPAY_API_KEY,
  network: 'base'
})

// Create a bounty
await client.createBounty({
  repository: 'owner/repo',
  issueNumber: 123,
  amount: 100 // USDC
})
```

## Smart Contract Integration

For products with smart contracts, connect your wallet and interact:

```typescript
import { useWriteContract } from 'wagmi'
import { parseEther } from 'viem'

const { writeContract } = useWriteContract()

await writeContract({
  address: contractAddress,
  abi: BountyABI,
  functionName: 'createBounty',
  args: [payeeAddress, parseEther('1')],
  value: parseEther('1')
})
```
