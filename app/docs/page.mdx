# Lucci Documentation

Lucci is the economic layer for open source. We provide tools to pay contributors in stablecoins and automate everything outside the repo so developers can focus on code.

## Our Products

### BountyPay

Pay contributors in stablecoins for completed work. Fund with USDC on Base and let BountyPay handle automated payments when issues are resolved.

**Status:** Alpha

[View Documentation →](/docs/products/bountypay)

### Pilot

An AI agent that automates everything outside the repo—marketing, coordination, token allocation—so developers can focus on code.

**Status:** Coming Soon

[View Documentation →](/docs/products/pilot)
